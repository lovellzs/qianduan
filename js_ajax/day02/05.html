微博接口调试综合案例

01 基本说明

01 整个页面主要由三部分组成：文本输入框 + 已发布微博列表 + 页码
02 页面的结构和CSS文件已经提供，把重点放在业务逻辑和网络接口调用上面
03 发送网络请求的时候使用jQuery框架中的ajax方法
04 需要实现的功能和调试的接口有：
（1）发布微博数据
（2）显示发布的微博数据
（3）显示默认已经发布的所有微博数据
（4）显示微博页码
（5）点击微博页码可以查看对应的微博数据
（6）点赞|踩|删除微博功能

02 发布微博数据核心和基本实现

基本过程：

01 在DOM加载完毕之后，先获取输入文本框和提交按钮
02 检查发布微博数据的接口文档
03 为按钮添加点击事件
04 在按钮的点击事件中使用jQuery发送GET请求，点击按钮之后获取文本输入框中的内容保存作为请求的参数
05 在请求成功的回调方法中打印响应体和接口文档进行比对

核心实现：

    $(function () {

       // 01 获取页面中的指定标签
        var oText = $("#submitText");
        var oSubmitBtn = $("#btn_send");

        //02 查看发送请求的接口信息：
        //发送一条微博数据接口
        //请求路径：weibo.php?act=add&content=xxx
        //返回内容：{error:0, id: 新添加内容的ID, time: 添加时间}

        //03 为按钮添加点击事件，发送请求
        oSubmitBtn.click(function () {
            $.ajax({
                "url":"weibo.php?act=add&content=" + oText.val(),
                "type":"GET",
                "success":function (xmlHttp) {
                    console.log("成功" + xmlHttp);

                    //成功{error: 0, id: 11, time: 1493605104}
                    //第一个参数：error为0表示请求成功
                    //第二个参数：这一条微博具体的id编号
                    //第三个参数：发送这条微博的时间戳
                },
                "error":function (code) {
                    console.log("失败" + code);
                }
            })
        })

    });

03 显示微博数据

基本思路：

01 请求成功之后先清空文本输入框的内容
02 处理要具体显示的内容
    001 文本输入框中发送请求时提交的内容
    002 响应体中获取的特定格式时间处理（timeFormatter方法）
03 拼接代码片段（模板）
04 创建临时标签（把代码片段设置为标签的innerHTML）
05 把创建好的标签通过append方法添加到指定的标签中

核心实现：

    $.ajax({
        "url":"weibo.php?act=add&content=" + submitText,
        "type":"GET",
        "success":function (xmlHttp) {
            //处理JSON数据
            var result = eval("(" + xmlHttp + ")");
            //000 发送完网络请求之后清空内容
            oText.val("");  //清空

            //001 获取要显示的内容：内容（submitText）和时间（timeText）
            var timeM = result["time"];
            var time = dateFormatter(timeM);

            //002 根据内容来设置标签
            var html = '' +
                '<div class="reply">'+
                '<p class="replyContent">'+submitText+'</p>'+
                '<p class="operation">'+
                '<span class="replyTime">'+time+'</span>'+
                '<span class="handle">'+
                '<a href="javascript:;" class="top">0</a>'+
                '<a href="javascript:;" class="down_icon">0</a>'+
                '<a href="javascript:;" class="cut">删除</a>'+
                '</span>'+
                '</p>'+
                '</div>';

            //003 创建标签
            var tempDiv = document.createElement("div");
            tempDiv.innerHTML = html;
            //004 显示在页面中
            oMsgList.append(tempDiv);
        }

时间处理函数：

    function dateFormatter(time) {

        //01 先根据传入的时间戳创建date
        //注意：时间戳需要*1000 再作为参数进行处理
        var date = new Date(time * 1000);

        //02 处理格式
        var arrM = [];
        //arrM.push(date.getYear()+"-");
        arrM.push(date.getFullYear()+"-");
        arrM.push(date.getMonth()+1 + "-");
        arrM.push(date.getDay()+" ");
        arrM.push(date.getHours()+":");
        arrM.push(date.getMinutes()+":");
        arrM.push(date.getSeconds());
        return arrM.join("");
    }

05 微博列表优化~②

优化思路：

抽取创建微博标签的内容为单独的方法
该方法的主要内容：
    （1）先创建空的div
    （2）为div添加class
    （3）获取标签内部需要用到的内容（时间 + 文本）
    （4）拼接HTML代码片段
    （5）调用html()来把HTML代码片段写入到div标签中
    （6）把创建好的div标签返回

核心代码：

    function createEle(obj) {
        //01 创建空的div
        var tmpDiv = $("<div></div>");
        //02 为div添加class
        tmpDiv.addClass("reply");
        //03 设置div的内容
        var time = dateFormatter(obj["time"]);
        var html = '' +
            '<p class="replyContent">'+obj.submitText+'</p>'+
            '<p class="operation">'+
            '<span class="replyTime">'+time+'</span>'+
            '<span class="handle">'+
            '<a href="javascript:;" class="top">0</a>'+
            '<a href="javascript:;" class="down_icon">0</a>'+
            '<a href="javascript:;" class="cut">删除</a>'+
            '</span>'+
            '</p>';
        tmpDiv.html(html);
        return tmpDiv;
    }

06 显示默认的微博数据

处理思路：

（1）发布了多条微博数据，重新刷新后没有数据
（2）页面加载完应该马上发送请求获取并显示第一页数据
（3）请求某页数据的接口和实现
（4）获取返回的json数据转换为数组（数组，使用eval不需要加圆括号）
（5）遍历创建div并append到指定的标签

主要任务：

（1）封装函数getDefaultMsg用于发送请求获取第一页的微博数据
（2）遍历请求得到的对象数组并创建微博标签添加到指定的标签中
（3）在页面加载完成后调用该方法

核心代码：

        getDefaultMsg();
        function getDefaultMsg() {
            $.ajax({
                "url":baseURL,
                "data":"act=get&page=1",
                "type":"GET",
                "success":function (response) {
                    //01 把获取数据转换为js的对象
                    var obj = eval(response);
                    //02 遍历获取的对象数组
                    $.each(obj,function (key,value) {
                        //03 根据每条数据创建div标签并添加
                        var div  = createEle(value);
                        oMsgList.append(div);
                    })
                }});
        }

07 获取并显示微博数据的页码

处理思路：

（1）查看获取微博页码的接口：weibo.php?act=get_page_count
（2）封装方法getPageCount，发送请求获取页面的数量
（3）在请求成功的回调中使用循环创建页码并显示
    001 获取微博页码的数量
    002 使用for循环遍历
    003 在for循环中把表示页码的代码片段包装成jQuery实例对象
    004 使用append方法把每个页码标签都添加到指定的标签中
    005 设置第一页为默认展示的页面（添加样式class）

主要任务：

（1）封装获取微博数据页码的函数getPageCount
（2）在页面加载完毕后就调用执行一次该方法

核心实现：

    function getPageCount() {
        $.ajax({
            "url":baseURL,
            "data":"act=get_page_count",
            "type":"GET",
            "success":function (response) {
                //使用for循环进行遍历创建相应的页码标签
                var count = eval("("+response+")")["count"];
                for(var i = 0;i<count;i++)
                {
                    var page = $('<a href="javascript:;">'+(i+1)+'</a>');
                     oPage.append(page)
                }

                oPage.children().first().addClass("active");
            }});
    }

08 显示对应页码的微博

处理思路：

(1) 为所有的页码都添加点击事件
    注意：需要先把page包装成jQuery对象然后传递给封装好的函数作为参数
    注意：获取page标签内容的时候使用page.text()而不是page.text
(2) 在点击事件中发送网络请求获取当前页码对应的数据
    1) 复用获取默认页码数据的方法，传递参数（index）
    2) 点击按钮之后会把请求得到的数据和当前已有的数据拼接在一起，应当先删除再处理
    3）点击页码标签应该切换页码的选中状态
        page.addClass("active").siblings().removeClass("active");
(3) 发布微博数据之后，页码信息应该更新
    1）在发布微博数据之后，应该调用获取页码方法重新显示最新的页码信息
    2）在getPageCount方法中，在请求成功的回调中应该先把之前的页码信息删除（清空），然后在创建标签并添加在指定的div标签中

核心代码

    function getPageCount() {
            $.ajax({
                "url":baseURL,
                "data":"act=get_page_count",
                "type":"GET",
                "success":function (response) {

                    //请求成功之后，先把以前的页码信息清空
                    oPage.html("");

                    //使用for循环进行遍历创建相应的页码标签
                    var count = eval("("+response+")")["count"];
                    for(var i = 0;i<count;i++)
                    {
                         var page = $('<a href="javascript:;">'+(i+1)+'</a>');
                        // page = '<a href="javascript:;">'+(i+1)+'</a>';
                         oPage.append(page);
                        addPageClickEvent(page);
                    }

                    oPage.children().first().addClass("active");
                }});
        }

    function addPageClickEvent(page) {
            page.click(function () {
                //alert(page.text());
                //切换页码的选中状态
                //siblings 找到page元素的所有同胞元素，为当前的标签添加class删除其他标签中的class
                page.addClass("active").siblings().removeClass("active");
                getMsgWithPage(page.text());
            })
        }

清空操作相关代码：

（1）oMsgList.html("");  //在getMsgWithPage方法成功回调中清空
（2）oPage.html("");     //在getPageCount方法成功回调中清空

09 实现点赞和踩的功能

实现思路：

（1）要实现点赞和踩的功能，需要给点赞，踩等按钮添加点击事件
（2）在写入微博内容之后，先找到对应的标签（通过div.find("class")）;
（3）抽取方法addTopClickEvent|addDownClickEvent等,把查找到的标签作为参数传递给该方法
（4）说明顶和踩等操作都需要发送网络请求，演示终极版本的页面说明
（5）查看对应的接口，说明参数中用到了id属性，那么封装好的方法中就需要多一个参数（obj对象）
（6）发送点赞的网络请求，说明请求成功之后服务器返回的结果是{error:0},没有意义，我们需要做的就是请求成功之后+1
（7）同样的思路处理踩的功能（直接拷贝修改请求路径）

核心代码：

    function addTopClickEvent(dom,obj) {
            //添加点击事件
            dom.click(function () {

                //发送网络请求
                $.ajax({
                    "url":baseURL,
                    "data":"act=acc&id="+obj.id,
                    "type":"GET",
                    "success":function (response) {
                        dom.text(parseInt(dom.text()) + 1);
                    }});
            });
        }

    function addDownClickEvent(dom,obj) {
            //添加点击事件
            dom.click(function () {

                //发送网络请求
                $.ajax({
                    "url":baseURL,
                    "data":"act=ref&id="+obj.id,
                    "type":"GET",
                    "success":function (response) {
                        dom.text(parseInt(dom.text()) + 1);
                    }});
            });
        }
