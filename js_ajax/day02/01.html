知识点回顾

001 基本概念：客户端|服务器|请求|响应|URL

    客户端：app|浏览器|微信H5端
    服务器：为客户端提供数据或资源的电脑
    请求：客户端向服务器端索要数据的行为
    响应：服务器端向客户端返回数据的行为
    URL:统一资源定位符，使用URL可以在互联网中找到唯一的服务器，类似于生活中的身份证号码

002 通信过程：

    （1）请求：索要数据
         请求头：对客户端以及请求本身的描述信息
         请求体：POST请求中把参数放在请求体中传递

    （2）响应：返回数据
        响应头：对服务端以及响应本身的描述信息
        响应体：返回的具体数据
        状态行：响应状态码 + 原因短语

003 发送请求的方式：

    GET请求：参数直接拼接在URL后面，有长度限制，不安全。
    POST请求：参数放在请求体重传递，没有长度限制，相对安全。

004 使用表单发送请求知识点

    （1）通过action来设置请求路径
    （2）通过method来设置请求方法，默认发送的是GET请求
    （3）如果需要提交参数给服务器端，那么每个提交的参数都必须有对应的key,这个key通过往表单中添加name属性来设置
    （4）使用表单进行文件上传注意点
        1）请求方式必须是post请求
        2）必须设置enctype属性
        3）文件上传到服务端之后默认保存在临时路径，如需要保存那么请剪切到安全的路径
        4）大文件上传需要对服务器端的php.ini文件进行配置

    （5）代码示例：
        <form action="请求路径" method="get">
        <input type="text" placeholder="请输入用户名" name="username">
        <input type="text" placeholder="请输入密码" name="password">
        <input type="submit">
        </form>

005 使用ajax发送GET请求

    （1）ajax的特点：异步处理请求任务
    （2）基本过程：
        1）创建请求对象（XMLHTTPRequest）[兼容性处理]
        2）设置请求路径 open（请求方法，请求路径，是否异步）
        3）发送网络请求 send
        4）监听网络请求的状态（onreadystatechange）
        5）处理服务器返回的数据

    （3）示例代码

        //001 创建请求对象
        var request = new XMLHttpRequest();

        //002 设置请求方法和路径
        request.open("get","请求路径",true);

        //003 发送请求
        request.send();

        //004 监听请求发送状态
        request.onreadystatechange = function () {
            //如果接收到服务器的反馈,那么就处理响应体
            if (request.readyState == 4)
            {
                //005 处理服务器的返回值
                if (request.status >=200 && request.status<300 || request.status == 304)
                {
                    console.log("请求成功");
                }else
                {
                    console.log("请求失败");
                }
            }
        }

006 使用ajax发送POST请求

    （1）基本过程
        1）创建请求对象（XMLHTTPRequest）[兼容性处理]
        2）设置请求路径 open（请求方法post，请求路径，是否异步）
        +) 设置请求头信息 setRequestHeader(Content-type:...)
        3）发送网络请求 send [把参数拼接为指定格式的字符串作为send方法的参数]
        4）监听网络请求的状态（onreadystatechange）
        5）处理服务器返回的数据

    （2）示例代码（核心代码）
        // 设置请求方法和请求路径
        xmlHttp.open("post",url,true);
        //设置请求头
        xmlHttp.setRequestHeader("Content-type","application/x-www-form-urlencoded");
        //发送请求
